# Sistema de Gestión de Préstamos de Libros

## 📌 Descripción
Este proyecto permite gestionar préstamos de libros utilizando Java y programación orientada a objetos.

## ⚙️ Funcionalidades
- Registro de libros
- Registro de usuarios
- Préstamos
- Devoluciones

## 🚀 Ejecución
1. Abrir el proyecto en NetBeans
2. Ejecutar la clase principal

## 🧠 Tecnologías
- Java
- POO (Encapsulamiento, Herencia y Polimorfismo)
