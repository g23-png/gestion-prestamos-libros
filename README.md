# 📚 Sistema de Gestión de Préstamos de Libros

## 📌 Descripción
Este proyecto consiste en el desarrollo de un sistema para la gestión de préstamos de libros, implementado en Java bajo el paradigma de Programación Orientada a Objetos (POO).

El sistema permite administrar libros, usuarios, préstamos y devoluciones, facilitando el control y organización de una biblioteca de manera eficiente.

---

## 🎯 Objetivo
Desarrollar una solución que permita gestionar el préstamo de libros mediante una estructura modular, aplicando principios de programación orientada a objetos como encapsulamiento, herencia y polimorfismo.

---

## ⚙️ Funcionalidades
- 📖 Registro de libros  
- 👤 Registro de usuarios  
- 🔄 Gestión de préstamos  
- ✅ Gestión de devoluciones  
- 📊 Control de disponibilidad de libros  

---

## 🧠 Principios de Programación Orientada a Objetos

### 🔹 Encapsulamiento
Se implementa mediante atributos privados y métodos públicos (getters y setters), garantizando la protección de los datos.

### 🔹 Herencia
Se utiliza una clase base `Persona`, de la cual heredan otras clases como `Usuario`, permitiendo la reutilización de atributos comunes.

### 🔹 Polimorfismo
Se aplica mediante la sobrescritura de métodos (por ejemplo, `toString()` o métodos de comportamiento), permitiendo diferentes implementaciones según la clase.
